# Palindromes

## Description
A console program that takes input from the user, and checks to see if the words or phrase is a palindrome. 

Examples	
--------
| Input                              |  Output |
|------------------------------------|---------|
| Stars                              |  False  |
| O, a kak Uwakov lil vo kawu kakao! | True    |
| Some men interpret nine memos      | True    |
